# cord

a lightweight message relay that mirrors messages between slack and discord text channels in real time, syncing conversations both ways without workflows, dashboards or unnecessary automation

## setup

1. visit [cord.zany.digital](https://cord.zany.digital)
2. connect your slack workspace
3. connect your discord server
4. map channels
5. chat away

> [!IMPORTANT]
> make sure to run `/invite @cord` on any mapped slack channels
